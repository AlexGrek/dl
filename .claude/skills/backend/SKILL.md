## Route Map

| Pattern | Auth | Handler | File |
|---|---|---|---|
| `GET /` | none | Preact SPA (React Router fallback) | `main.go` |
| `GET /d/{path...}` | none | Stream file from WebDAV upstream | `download.go` |
| `GET /rs/{path...}` | none | Stream release file from WebDAV upstream | `download.go` |

| `POST /api/v1/auth/keys` | Bearer master key | Create API key | `auth.go` |

| `PUT /api/v1/release/{bucket}/{os_arch}/{file...}` | Bearer JWT (`release-write:{bucket}`) | Upload release file | `release.go` |

## Auth Flow

```
Master key (from .secrets.yaml)
  ‚îÇ
  ‚îú‚îÄ POST /api/v1/auth/keys       ‚Üí create scoped API key (dlk_...)
  ‚îÇ
  ‚îî‚îÄ POST /api/v1/auth/token      ‚Üí 1-hour JWT (master gets read+write+release-create)
         ‚Üë
  Any API key also hits this endpoint
         ‚îÇ
         ‚îî‚îÄ JWT used as Bearer on all protected endpoints
```

**The master key is never stored in BoltDB.** It is compared directly (string equality) from the in-memory config.

### JWT Claims (`src/auth.go ‚Üí Claims`)

```go
type Claims struct {
    jwt.RegisteredClaims
    KeyID   string   `json:"kid"`
    Scopes  []string `json:"scopes"`
    RootDir string   `json:"root_dir,omitempty"`
}
```

- `ExpiresAt`: now + 1 hour
- Algorithm: HS256, secret from `cfg.JWTSecret`

### Scopes (encoded in JWT)

| Scope | Grants |
|---|---|
| `read` | Read via WebDAV proxy (restricted to `root_dir` if set) |
| `write` | Write via WebDAV proxy (restricted to `root_dir` if set) |
| `release-create` | Create new release buckets (MKCOL on `/rs/`) |
| `release-write:{bucket}` | Upload to `/rs/{bucket}/...` specifically |

## BoltDB Schema (`src/store.go`)

**Bucket:** `apikeys`

- **Key:** `hex(sha256(raw_api_key))` ‚Äî hashed for storage, never stored in plaintext
- **Value:** JSON-encoded `APIKey` struct

```go
type APIKey struct {
    ID          string    `json:"id"`          // first 12 chars of raw key
    Description string    `json:"description"`
    Scopes      []string  `json:"scopes"`
    RootDir     string    `json:"root_dir,omitempty"`
    CreatedAt   time.Time `json:"created_at"`
}
```

**API key format:** `dlk_<32-byte base64url>` ‚Äî generated by `crypto/rand`, prefixed `dlk_`.

## Config (`.secrets.yaml`)

App reads a flat YAML file ‚Äî **not** a Kubernetes Secret manifest.

```yaml
webdav_url: "https://host"       # required ‚Äî must include scheme
webdav_username: "user"          # required
webdav_password: "pass"          # required
master_key: "long-random-string" # required
jwt_secret: "long-random-string" # required
db_path: "./dl.db"               # optional, default: ./dl.db
port: "8080"                     # optional, default: 8080
```

Loaded by `loadConfig(path string) (*Config, error)` in `src/config.go`. Passed to binary via `-secrets` flag (default `.secrets.yaml`).

## WebDAV Proxy Details (`src/webdav.go`)

- Uses `httputil.NewSingleHostReverseProxy` pointing at `cfg.WebDAVURL`
- Director strips `/api/v1/wd` prefix, injects `Authorization: Basic` for upstream
- All HTTP methods pass through (including PROPFIND, MKCOL, COPY, MOVE, etc.)
- Scope enforcement: read-only methods need `read` or `write`; mutating methods need `write`
- Path restriction: if `TokenInfo.RootDir != ""`, bare path must have it as prefix

## Release Upload Details (`src/release.go`)

- `POST /api/v1/release/create` body: `{"bucket":"name"}` ‚Üí MKCOL `/rs/` then `/rs/{bucket}`
- `PUT /api/v1/release/{bucket}/{os_arch}/{file...}` ‚Üí MKCOL intermediate dirs, then PUT to WebDAV at `/rs/{bucket}/{os_arch}/{file}`
- Scope check: `release-write:{bucket}` OR `write` scope accepted
- `webdavMKCOL` treats 201, 405, 207 as success (directory may already exist)

## Download Details (`src/download.go`)

- `GET /d/{path...}` ‚Üí upstream `/{path}` (no auth, path cannot contain `..`)
- `GET /rs/{path...}` ‚Üí upstream `/rs/{path}` (no auth)
- Forwards: `Range`, `If-None-Match`, `If-Modified-Since` request headers
- Forwards: `Content-Type`, `Content-Length`, `Content-Disposition`, `Last-Modified`, `ETag`, `Accept-Ranges`, `Content-Range` response headers
- Streams with `io.Copy` ‚Äî no buffering

## Conventions

| Do | Don't |
|---|---|
| Stream file responses with `io.Copy` | Buffer entire files in memory |
| Use typed `ctxKey` for context values | Use plain string context keys |
| Store scopes in JWT claims | Re-query DB on every request |
| Return `403` for valid token, wrong scope | Return `401` for scope mismatch |
| Use `go:embed all:static` for frontend | Serve frontend from disk in production |
| Use external HTTP routers (chi, echo, gin) when routes/middleware get complex | Add them just for a simple WebDAV proxy or key exchange |
| Plain text errors via `http.Error` | JSON envelope for errors |
| Inject Basic Auth in proxy Director | Pass credentials in the URL |
